This program can be used to find Waldo in greyscale images. 
The program calculates the sum of the absolute differences (SAD)
between the RGB values in the scene image and in the template image and
once a match has been found the program draws a white box around Waldo.
The file Waldo.java conatins the code to find Waldo and the file WaldoTester.java is the tester class that accompanies it.
The file waldo.png is an example image used and the file WaldoBoxed.jpg is the output iamge with Waldo being boxed in white.
